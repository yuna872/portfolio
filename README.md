# Portfolio

프론트엔드 개발자 김유나의 포트폴리오 웹사이트

🔗 **https://portfolio-yuna-front.vercel.app**

## Tech Stack

- **Framework**: Next.js 16, React 19
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4
- **Deployment**: Vercel

## Sections

| 섹션 | 설명 |
|------|------|
| **About** | 자기소개 및 핵심 역량 |
| **Career** | 경력 사항 및 프로젝트 상세 (아코디언 UI) |
| **Projects** | 개인/외주 프로젝트 (hover 펼침 인터랙션) |
| **Education** | 교육·스킬·자격증·어학·학력 통합 |
| **Contact** | 이메일 및 GitHub 링크 |

## Getting Started

```bash
npm install
npm run dev
```

## Deployment

Vercel + GitHub Actions 기반 자동 배포
